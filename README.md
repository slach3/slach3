# GameNews - Portal de Notícias de Jogos

Um portal que agrega automaticamente as últimas notícias do mundo dos jogos eletrônicos de várias fontes confiáveis.

## Sobre o Projeto

Este projeto é um site de notícias de jogos chamado **GameNews**, com as seguintes características:
- Agregação automática de notícias via RSS
- Filtros por categoria
- Layout limpo e funcional
- Design responsivo

O objetivo é manter uma plataforma simples e eficiente para os entusiastas de jogos eletrônicos acompanharem as últimas novidades do setor.

## Estrutura do Projeto

- `scraper.py`: Script Python para coleta automática de notícias
- `noticias.js` e `noticias.json`: Manipulação e armazenamento das notícias
- `atualizar_noticias.bat`: Script para automação da atualização de notícias
- `docs/`: Pasta contendo o site publicado
